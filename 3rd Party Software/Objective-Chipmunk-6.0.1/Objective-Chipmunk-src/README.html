How to Get Started:

Installing Objective-Chipmunk is as simple as dropping the Objective-Chipmunk-iPhone/ directory into your Xcode project. At that point you should simply be able use #import "ObjectiveChipmunk.h"

Where to Go Next:

  • Skim the Chipmunk Physics documentation. ( local | online) It's C API centric, but Chipmunk was programmed in an Object-Oriented style so the concepts apply directly to the Objective-C API.
  • Read the Objective-C API Docs: (local | online)
  • Get sample code from the Chipmunk downloads page.

If you need more example code, I've also included the source code for the Chipmunk Physics iPhone Demo App (iPhoneChipmunk) as well as Snap Physics (iPhoneSnap).

Building Objective-Chipmunk:

While Objective-Chipmunk comes prebuilt, we also believe you bought the right to the full source code for it by supporting us. That way if you want to make any modifications or need to debug something you can.

The easiest way to build Objective-Chipmunk is to run the iphonestatic.command in the Objective-Chipmunk directory (double click in the Finder or run in the Terminal). This will build the iPhone static library will all the optimizations turned on, and build a simulator build with all the debug options turned on. It also builds a simulator only version.

There is also an Xcode project provided. The iphonestatic.command script actually uses this same project file.
